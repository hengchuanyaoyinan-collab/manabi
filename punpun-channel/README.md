# ぷんぷんの世事見聞録 - 完全自動化プロジェクト

YouTubeチャンネル「ぷんぷんの世事見聞録」の動画制作〜投稿までを完全自動化するシステム。

> このフォルダは manabi リポジトリとは別プロジェクトです。将来的に独立リポジトリへ分離予定。

## 🎯 ビジョン

### 目標
- **第1段階**: 専業収益化（登録者10万〜30万人、月20万以上の広告収入）
- **第2段階**: 登録者100万人、キャラクターIP展開

### 戦略
競合「ぴよぴーよ速報」（108万登録者）の投稿停滞を好機と捉え、類似フォーマットで高品質な動画を毎日投稿することで、同じアルゴリズム推薦枠を取りにいく。

### 差別化
- **真面目7 : エンタメ3** のハイブリッド路線
- ぴよぴーよ（砕けすぎ）と「大人の学び直しTV」（固すぎ）の中間ポジション
- 「やばエピソード」重視で重いテーマも親しみやすく

## 🛠 技術スタック

| 工程 | ツール | 費用 |
|---|---|---|
| 台本生成 | Claude Code (Max契約) | $0（契約内） |
| 音声合成 | VOICEVOX | $0 |
| 字幕同期 | Whisper | $0〜$2/月 |
| 動画組立 | Remotion | $0 |
| 画像取得 | Wikipedia API / いらすとや | $0 |
| エンコード | FFmpeg | $0 |
| 自動投稿 | YouTube Data API v3 | $0 |

**月額想定コスト: 0〜500円**
