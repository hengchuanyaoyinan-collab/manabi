# ビジョン・戦略

## チャンネルコンセプト

**「学校で教えてくれない、歴史の裏側とやばい話」**

歴史上の人物・出来事について、教科書には載らない「やばいエピソード」「狂気」「人間臭さ」を中心に、20分でじっくり解説するチャンネル。

## ターゲット視聴者

### メインターゲット
- 20代〜40代男女
- 歴史に興味はあるが、固い解説は苦手
- 通勤・家事中の「ながら視聴」層
- ぴよぴーよ速報・大人の学び直しTV視聴者

### サブターゲット
- 歴史好きだが、教科書的解説に飽きている層
- 「世界史B」未履修で海外の歴史に薄い日本人

## ポジショニング

```
         真面目／学問的
              ▲
              │
        大人の学び直しTV
              │
              │
    ★ ぷんぷんの世事見聞録 ← ここを狙う
              │

              │
              ▼
         エンタメ／軽い
```

「ちゃんと学べるけど、重くない」中間ゾーンが空白。

## 競合分析（詳細は 04-competitor-analysis.md）

### ぴよぴーよ速報
- 登録者 108万、動画 228本
- 最新動画から1ヶ月経過（投稿ペース鈍化）
- → **視聴者が新コンテンツに飢えているタイミング**
- 同フォーマットで高品質動画を供給すれば、アルゴリズム推薦を獲得可能

### 大人の学び直しTV
- 真面目な歴史解説
- 質は高いが、一般層には難しい
- 差別化: ぷんぷんはもっと噛み砕く、エピソード重視

## 差別化の柱

### 1. 「やばエピソード」重視の切り口
教科書が避ける「狂気」「スキャンダル」「タブー」を正面から扱う。
ただし単なるゴシップにせず、歴史的文脈の中で語る。

### 2. キャラクター性
ぷんぷん（白丸顔＋赤い唇）を前面に。将来的にスタンプ・グッズ化。

### 3. 毎日投稿
競合が週1〜2本なのに対し、毎日投稿で供給量勝負。
完全自動化により実現。

### 4. 20分尺
- 4分だと深掘りできない
- 30分超は視聴維持率が下がる
- 20分は広告収益効率も良い（中盤広告3〜4個挿入可）

## 題材選定の方針

### 優先順位

**A. 最優先（ぴよぴーよ未開拓 × やばエピソード濃い）**
- エリザベート・バートリ（血の伯爵夫人）
- カリグラ帝（ローマ狂気皇帝）
- 則天武后（中国唯一の女帝、粛清の鬼）
- エカチェリーナ2世（スキャンダル満載）
- アブドゥル・ハミト2世（赤いスルタン）
- カリグラ、ネロ、ドミティアヌス（ローマ暴君シリーズ）
- 西太后（清末の狂気）

**B. 定番（ぴよぴーよと被るが需要大）**
- チンギスハン（既に投稿済み、リメイク可）
- 織田信長（既に投稿済み、リメイク可）
- ナポレオン
- 毛沢東
- アレクサンドロス大王

**C. テーマ系（人物以外）**
- 中国の歴史（王朝ごと）
- 戦争の歴史（第二次大戦、冷戦）
- 宗教の歴史（キリスト教・イスラム教の闇）

### 避けるべき題材
- 現代日本政治（炎上リスク）
- 特定の宗教批判に偏るもの
- 未解明の陰謀論
