### 仮説: 投稿ペース鈍化の理由

複数の可能性:
1. **複数チャンネル運営**: 別チャンネルに労力分散
2. **クオリティ追求**: 長尺動画 (5 時間超) に時間を取られている
3. **SHORTS シフト**: ショート動画に注力している可能性
4. **収益化に満足**: 既存ライブラリの広告収益で十分回っている

→ **どの理由でもアルゴリズム推薦の空白は生まれている。**
→ 我々はここを狙う。

## 4. アルゴリズム視点の戦略

### YouTube 推薦アルゴリズムの特性
1. **同フォーマット類似度**: ぴよぴーよ視聴者には同フォーマットの動画を推薦
2. **公開直後の伸び**: 最初の 1〜2 時間の CTR・視聴維持率が決定的
3. **シリーズ化**: 同じチャンネルの動画が連続再生されると評価UP
4. **長時間視聴**: 20 分動画 × 高維持率 = 強い

### 我々の戦術
- ぴよぴーよと**同じフォーマット**を採用 (推薦枠を奪う)
- **毎日投稿**で「最新動画」枠を独占
- **シリーズ化**で連続再生を促す
- **20 分** + 高維持率を狙う構成
- **インパクトのあるサムネ** で CTR を上げる

## 6. リスク

### 競合リスク
- ぴよぴーよが投稿再開 -> 直接競合化
- 新規参入チャンネルが同戦略を取る
- → **対策**: 毎日投稿で先行者利益を取る

### YouTube ポリシーリスク
- 自動生成コンテンツへの規制強化 (特に 2026 年以降)
- 「Inauthentic content」判定で収益化停止
- → **対策**: AI 全面依存ではなくキャラ性で人格を立たせる、概要欄で透明に

### 著作権リスク
- いらすとや: 商用利用 OK だが利用ルール遵守
- Wikipedia 画像: ライセンス確認必須 (CC-BY-SA / Public Domain など)
- BGM: 必ずロイヤリティフリー or YouTube オーディオライブラリから
- → **対策**: 取得ソースとライセンスを記録 (config/asset-credits.json)

### 炎上リスク
- 残虐エピソードの過剰演出
- 特定民族・宗教への配慮不足
- 歴史解釈の偏り
- → **対策**: 台本生成プロンプトで「諸説あり」を必須化、最初の 1 ヶ月は手動レビュー
