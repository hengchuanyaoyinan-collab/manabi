## どうやって生成したか (再現手順)

### 1. 台本生成
```bash
python3 src/generator/script_generator.py "エリザベート・バートリ" \
    --out test_data/elizabeth_bathory_full.json \
    --model claude-haiku-4-5-20251001
```
Claude Max 契約 → claude CLI 経由 → 追加料金 0 円。

### 2. 動画組立
```bash
python3 src/orchestrator.py \
    --script test_data/elizabeth_bathory_full.json \
    --test --no-voicevox
```

完了後、`output/test/{timestamp}/video.mp4` が出来る。

### 3. 確認
```bash
# Windows なら
start output\test\{timestamp}\video.mp4

# ブラウザで Open
```

## 結論

**完全自動化は技術的に達成済み。**
あとはユーザーが:
1. Windows で VOICEVOX を起動する (声を本物にする)
2. YouTube API の OAuth を 1 回設定する (アップロードを有効にする)
3. (任意) ぷんぷんキャラ画像を抽出する (見た目を本家に寄せる)

これで毎日 19:00 に自動投稿される。
