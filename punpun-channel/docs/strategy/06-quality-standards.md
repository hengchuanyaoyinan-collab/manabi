# 🎯 品質基準 (Quality Standards)

**動画を「作れる」のではなく、「見られる」までが制作。** この基準を満たさない動画は投稿しない。

---

## 🚫 投稿拒否ライン (1 つでも該当したら投稿禁止)

### 冒頭 30 秒
- [ ] **最初の 3 秒**で視聴者の興味を引くフック (衝撃事実・疑問・煽り) がある
- [ ] **最初の 30 秒**に以下の要素 3 つが入っている:
  - 主役の最終的な末路 or 最大の狂気 (ネタバレ先出し)
  - 数字 (再生時間・被害者数・在位年数等の具体数字)
  - 視聴者への問いかけ (「知ってます?」「どう思います?」)

### 脚本構造
- [ ] **3 分ごとに 1 つ** 驚きのエピソードがある
- [ ] **視聴者の予想を裏切る展開**が最低 3 回
- [ ] 情報羅列セクションが 45 秒以上連続していない
- [ ] セリフの**平均 24 字以下** (短文テンポ)
- [ ] 同じ感情 (emotion) が 5 シーン以上連続していない

### ビジュアル
- [ ] **地図** 最低 3 回以上 (時代背景を示す)
- [ ] **肖像画 or 写真** 5 回以上
- [ ] **いらすとや等のイラスト** 15 回以上 (抽象概念を可視化)
- [ ] **blank 背景** 5% 以下
- [ ] ぷんぷんの**表情が 4 種類以上**登場
- [ ] 主要シーンで**文字テロップ強調** (キーワードを画面中央に拡大)

### 音響
- [ ] **BGM** が全編に流れる (章ごとに切り替え)
- [ ] **場面転換効果音** が最低 5 回
- [ ] **衝撃 SE** (ドン・シャキン・ヒュッ) が最低 3 回
- [ ] 無音区間が 3 秒以上連続しない

### メタデータ
- [ ] タイトルが **30〜45 字** で、数字+煽りワード 2 個以上
- [ ] サムネが **1280x720** で極太フォント・3 色以内
- [ ] 概要欄に章タイムスタンプ + 参考資料
- [ ] タグ 15 個 (高競合 5 + 中競合 5 + 低競合 5)

### 炎上回避
- [ ] 現代政治人物への直接言及なし
- [ ] 特定民族・宗教への侮辱表現なし
- [ ] 「諸説あり」を最低 1 回挟む (歴史の不確実性を示す)
- [ ] 残酷描写は抽象化 (「惨殺された」OK、「首を切った」は NG)

---

## 📊 推奨値 (満たせばベター)

| 項目 | 推奨値 |
|---|---|
| 動画長 | 15〜22 分 (広告収益最大化) |
| シーン数 | 80〜120 |
| 平均シーン長 | 8〜12 秒 |
| 章数 | 7 章 |
| タグ文字数合計 | 300〜450 |
| 説明欄文字数 | 600〜1500 |

## 🎬 editing 標準

### テロップ
- **重要キーワード**は画面中央に 2〜3 秒間拡大表示
- フォント: 極太ゴシック、縁取り黒
- 色: 白 (基本)、黄 (強調)、赤 (衝撃)
- サイズ: 画面高の 1/10 以上

### カットテンポ
- オープニング: 1.5〜2.5 秒/カット
- 本編通常: 3〜5 秒/カット
- クライマックス: 1〜2 秒/カット
- まとめ: 4〜6 秒/カット

### トランジション
- 章変わり: フェード or カラーワイプ (1 秒)
- シーン変わり: カット or 短フェード (0.15 秒)
- 衝撃シーン: フラッシュ (0.1 秒)

### 音響配置
- BGM: 基本 -18dB、セリフ時 -24dB にダッキング
- SE: -6〜-3dB で目立たせる
- ナレーション: -6dB で聞き取りやすく

## 🔬 リサーチ基準

### 題材ごとに必須調査
1. 日本語 Wikipedia 全文 (一次)
2. 英語 Wikipedia (二次ソース。情報量多い)
3. 関連書籍 (最低 1 冊特定)
4. 最低 3 本の既存 YouTube 解説動画のタイトル+再生数
5. Google トレンドでの検索需要

### リサーチ不足の判定
以下があれば調査不足:
- 日付が曖昧 (「1500 年代」ではなく「1567 年 3 月」レベルまで)
- 人物関係が 5 人以下しか把握できてない
- 周辺情勢 (同時代の他国) が触れられていない
- 現代との比較・類推が 1 個も無い

## 🎨 キャラ再現基準

ユーザー提供素材と**以下の項目で同一** であること:
- [ ] 顔の形・比率
- [ ] 目の形・位置
- [ ] 口の形 (特に赤いリング形状)
- [ ] 眉毛の形
- [ ] 全体の色調

90% 以上一致しない場合、**ユーザー素材の入手を最優先**。

## ⚙️ 自動品質ゲート

`scripts/quality-audit.py` で台本を自動評価:
- スコア 70 点以下 → 投稿禁止、再生成
- スコア 70〜85 点 → 手動レビュー推奨
- スコア 85 点以上 → 投稿 OK

実行:
```bash
python scripts/quality-audit.py test_data/チンギスハン.json
```

---

## 📈 運営後の計測

投稿後 24 時間で以下を測定 (`daily-analytics.py`):
- CTR < 3% → サムネ問題、次回から再設計
- 維持率 < 30% → 脚本問題、構成見直し
- 平均視聴時間 < 半分 → 冒頭 30 秒のフック強化
- コメント 0 → 問いかけ不足、「どう思う?」を入れる

数値未達が 3 本連続したら、**品質基準自体を見直す**。

## 🎯 到達目標 (品質で比較できる競合)

**同カテゴリで当面目指すレベル:**
- ぴよぴーよ速報 (108 万登録、CTR 推定 5〜7%)
- 大人の学び直し TV
- ヘイカン (お笑い系歴史)

**ベンチマーク指標** (6 ヶ月以内):
- CTR: 4.5% 以上
- 平均視聴維持率: 40% 以上
- 平均視聴時間: 動画長の 45% 以上

---

**本基準は月 1 回見直す。達成できない項目は「なぜ」を分析して基準自体を改善する。**
