# Tan-i (α) — 単位取得レシピ

通信制大学の単位取得経験を、構造化レシピとして共有するプラットフォーム。
クックパッドの「料理レシピ」を「単位取得レシピ」に置き換えた発想。

このディレクトリは **`tani-app`** という独立した Vite プロジェクト。
親リポジトリ（`manabi`）とはコード的に独立しており、後で別リポジトリへの切り出しが可能。

## ローカル起動

```bash
cd tani-app
npm install
npm run dev
```

→ http://localhost:5174

## ビルド

```bash
npm run build
npm run preview
```

## デプロイ（Vercel）

Vercel で新プロジェクト作成時に Root Directory を `tani-app` に設定すると、
`manabi` 本体とは独立した URL でホストできる。

## 別リポジトリへの切り出し（後で）

```bash
git filter-repo --subdirectory-filter tani-app
```

で `tani-app` 配下だけを抜き出した新ブランチを作って、
新リポジトリに push すれば、履歴ごと独立できる。

## 設計思想

- **構造化UGC**：自由文ではなく、難易度・所要時間・取得方法をメタデータ化
- **検索価値**：科目別・難易度別・期間別で絞れる
- **累積資産化**：時間でデータが堀になる
- **誠実なコピー**：サンプルと実投稿を明確に区別、誇大表示しない
